# universal

Place-agnostic tools loaded on demand from the hub.

- **`mya_universal.luau`** — thin bootstrap that fetches **`games/MyaUniversal/init.lua`** and runs full Universal (ESP, combat helpers, fly/noclip, custom GUI stack). Sources live beside other games under **`games/MyaUniversal/`**.
- **`dumper.lua`** — standalone dump helper.

Document hub URLs in `Mya_Documentation/06_loader_and_hub.md`.
