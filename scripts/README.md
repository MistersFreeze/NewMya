# scripts

Build and dev helpers.

| Script | Role |
|--------|------|
| `build-loader-hub.ps1` | Runs `npm run build` in `packages/loader-hub` |
| `serve.ps1` | Optional: `python -m http.server` from repo root for local `MYA_LOCAL_ROOT` testing |

No amalgamation script in v1; use Luau `hub.luau` for static hosting.
